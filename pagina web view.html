Guia de HTML No1

  1. INTRODUCCION


    En esta guia aprenderemos los conceptos basicos de HTML.

    HTML (HyperText Markup Language) es el lenguaje de marcado estandar para crear paginas web.


  2. ELEMENTOS BASICOS


    Los elementos basicos de HTML incluyen:

    • Etiquetas

    • <h1> - Encabezado principal
      <h2> - Subencabezado
      <h3> - Encabezado de tercer nivel
      <h4> - Encabezado de cuarto nivel
      <h5> - Encabezado de quinto nivel
      <h6> - Encabezado de sexto nivel
      <h7> - Encabezado de septimo nivel
      <h8> - Encabezado de octavo nivel
      <h9> - Encabezado de noveno nivel
      <h10> - Encabezado de decimo nivel
      <h11> - Encabezado de undécimo nivel

      sigue asi hasta el h11

      <p> - párrafo
      <div> - contenedor
      <span> - contenedor en línea
      <header> - encabezado
      <footer> - pie de página
      <main> - contenido principal
      <section> - sección
      <article> - artículo
      <a> - para crear links

    • Atributos

      Los atributos son propiedades adicionales que se pueden agregar a las etiquetas HTML para proporcionar información adicional sobre el elemento.

      Por ejemplo:

      • href - Especifica la URL de un enlace.
      • src - Especifica la URL de una imagen.
      • alt - Proporciona un texto alternativo para una imagen.
      • class - Especifica una clase para aplicar estilos CSS.
      • id - Especifica un identificador único para un elemento.

    • Contenido

      El contenido de una página web se define dentro de las etiquetas HTML.

      Por ejemplo:

                  <h1>Título Principal</h1>
            <p>Este es un párrafo de texto.</p>
            <div>
                <h2>Subtítulo</h2>
                <p>Este es otro párrafo dentro de un contenedor.</p>
            </div>

    • Ejemplo de Estructura HTML

                  <!DOCTYPE html>
            <html>
            <head>
                <title>Mi Página Web</title>
                <link rel="stylesheet" href="estilos.css">
            </head>
            <body>
                <header>
                    <h1>Bienvenido a Mi Página Web</h1>
                </header>
                <main>
                    <section>
                        <h2>Sobre Mí</h2>
                        <p>Este es un párrafo sobre mí.</p>
                    </section>
                </main>
                <footer>
                    <p>© 2025 Mi Página Web</p>
                </footer>
            </body>
            </html>

    • como decorar una pagina web

      Para decorar una pagina web, podemos utilizar CSS (Cascading Style Sheets).

      CSS nos permite aplicar estilos a los elementos HTML, como colores, fuentes y layout.

      como implantar CSS en tu pagina web

      Para implantar CSS en tu pagina web, puedes utilizar la etiqueta <style> dentro de la sección <head> de tu documento HTML. Por ejemplo:

                  <style>
                body {
                    background-color: #f0f0f0;
                    font-family: Arial, sans-serif;
                }
                h1 {
                    color: #2c3e50;
                }
            </style>

      También puedes enlazar un archivo CSS externo usando la etiqueta <link>:

                  <link rel="stylesheet" href="estilos.css">
